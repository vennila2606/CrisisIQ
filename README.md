
# 🛰️ CrisisIQ: Crisis Intelligence & News Verification Environment

## 📌 Overview

CrisisIQ is an AI-based simulation environment designed to evaluate how intelligent agents make decisions during crisis situations.

In real-world disasters such as floods, earthquakes, or conflicts, misinformation spreads rapidly and can lead to panic or delayed responses. This project simulates such conditions and tests how an AI agent balances **speed and accuracy** when deciding how to act.

---

## 🚨 Problem Statement

During crises:
- Misinformation spreads quickly
- Fake alerts cause unnecessary panic
- Delayed action can cost lives

This project addresses the challenge of:
> Making reliable decisions under uncertainty and time pressure.

---

## 🎯 Objective

To build an environment where an AI agent:
- Receives uncertain and noisy information
- Decides appropriate actions
- Is evaluated based on correctness, speed, and risk

---

## 🧠 Key Features

-  Simulates real-world crisis scenarios
-  Balances **accuracy vs speed**
-  Handles **uncertainty and conflicting data**
-  Reward-based evaluation system
-  Multi-step decision making
